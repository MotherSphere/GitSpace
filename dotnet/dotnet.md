# .NET helper project

## Structure
```
dotnet/
└── GitSpace.Helper/
    ├── GitSpace.Helper.csproj
    └── Program.cs
```

## Build
From the repository root:
```
dotnet build dotnet/GitSpace.Helper/GitSpace.Helper.csproj
```

## Run
```
dotnet run --project dotnet/GitSpace.Helper -- "Hello from GitSpace"
```
